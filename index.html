Welcome to Cyber Defender!

How to Play:

What You've Learned

Phishing Detection

Always verify sender emails and be suspicious of urgent requests for personal information or unexpected links.

Password Security

Use strong, unique passwords for each account and enable two-factor authentication whenever possible.

Network Safety

Avoid unsecured public Wi-Fi networks and always use a VPN when accessing sensitive information in public.

General Security

Keep software updated, use antivirus protection, and verify website security before sharing sensitive information.

Cyber Safety Tips

  • Always verify the sender's email address before clicking links or downloading attachments
  • Look for HTTPS and the padlock icon in your browser before entering sensitive information
  • Use password managers to create and store complex passwords
  • Enable two-factor authentication on all accounts that support it
  • Keep your operating system and applications updated with the latest security patches
